# skills/hermespace

Public Hermes Agent skill for the Hermespace package.

Install:

```bash
mkdir -p "${HERMES_HOME:-$HOME/.hermes}/skills/hermespace"
cp -a skills/hermespace/. "${HERMES_HOME:-$HOME/.hermes}/skills/hermespace/"
# or symlink the directory for live checkout edits
```

SoT for agents: `SKILL.md`. Branch detail: `references/`.  
Package docs remain SoT for deep design (`INTEGRATION.md`, `docs/*`).
